<h1><b>Notes App</b></h1>h1>

A simple and interactive Notes Application that allows users to create and manage notes directly in the browser.
All notes are automatically saved using Local Storage, so data remains available even after refreshing or reopening the page.
This project demonstrates DOM manipulation, event handling, and browser storage using Vanilla JavaScript.

<hr>

🚀 Features

✅ Create new notes

✅ Edit existing notes

✅ Auto-save notes in Local Storage

✅ Persistent data (no database required)

✅ Delete notes functionality

✅ Clean and responsive UI

✅ Lightweight and fast

<hr>

🛠️ Tech Stack

HTML5
CSS3
JavaScript (Vanilla JS)

🧠 How It Works

Users create notes through the interface.
Notes are stored in the browser using Local Storage.
When the page reloads, saved notes are automatically loaded.
No backend or database is required.
